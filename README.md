# Selenium Automation Framework

This project is built using Selenium WebDriver, Java, TestNG, and Maven.

## Features

* Page Object Model (POM)
* Login automation
* Explicit waits
* Assertions and validations
* Cross-browser compatible structure

## Tools & Technologies

* Selenium WebDriver
* Java
* TestNG
* Maven
* IntelliJ IDEA

## Project Structure

src

* pages
* tests
* utilities
